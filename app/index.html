Obligatory First Post

I made a blog in JavaScript.

Why?

Why anything? I've been learning JavaScript for a couple of years now and felt it was about time I documented what I was working on, and shared my experiences.

I tried WordPress, but it has a lot of visual cruft and due to how widely used it is, it's basically a massive attack surface waiting to happen. I tried Tumblr too, but I also hate how it feels. I want to be able to display code snippets without screenshots, so building my own felt right. That's how I did when I learned ASP. 

alert('Hello World!')

How?

JavaScript and more JavaScript. The back-end is NodeJS with a MongoDB hosted by mLab. Initially the back-end was Go, but decided to move to JavaScript. Using Glitch as my editor has been loads of fun, and means that I can develop completely using my Raspberry Pi (which struggles with VS Code).

The front-end is AngularJS, with Showdown used to render MarkDown. HighlightJS is used to style code snippets with a GitHub(ish) theme.
